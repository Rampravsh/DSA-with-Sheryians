# Array Manipulation in JavaScript

This `app.js` file contains a collection of JavaScript code snippets that demonstrate various operations and manipulations on arrays.

## Features

The script includes examples for the following array operations:

- **Array Initialization**: Creating and initializing arrays.
- **User Input**: Populating an array with values entered by the user.
- **Sum of Elements**: Calculating the sum of all elements in an array.
- **Finding Extrema**:
    - Finding the maximum and minimum values in an array.
    - Finding the second largest and second smallest values.
- **Array Reversal**:
    - Reversing an array using an auxiliary array (with extra space).
    - Reversing an array in-place (without extra space).
- **Two-Pointer Technique**:
    - Segregating zeros and ones in an array.
    - Moving all negative numbers to one side of the array.

## Usage

Each section in `app.js` is a self-contained example. To run a specific example, you can uncomment the desired code block and execute the file using Node.js:

```bash
node app.js
```
